## dcat
A terminal-based 3D model viewer

## Usage
```bash
dcat --help
```

## Controls
- `q` - Quit the viewer
- `m` - Toggle wireframe
When `--fps-controls` is enabled:
- `w/a/s/d` - Move forward/left/backward/right
- `i/j/k/l` - Look around
- `Space` - Move up
- `Shift` - Move down
- `v` - Slow down
- `b` - Speed up
